Project Description

This project contains automated UI tests for the OpenCart e-commerce website.

The tests are created using Selenium WebDriver and TestNG following the Page Object Model (POM) design pattern.

Technologies Used-

-Java
-Selenium WebDriver
-TestNG
-Maven
-InteliJ IDEA
-GitHub



PROJECT STRUCTURE
The project includes automated tests for the following functionalities:

src/main/java/pages - Page Object Classes
src/test/java/tests - Test Classes
testng.xml - Test suite configuration
pom.xml - Maven dependencies
